# BananaLabs.gg

BananaLabs open-source organization site — Astro static.

## Features

- Live Sessions.gg status pill (fetched from Evolution API)
- Nav status alerts for service outages (mobile-responsive fixed strip)
- Contact form with honeypot bot protection and branded email routing
- Product showcase with ecosystem links

## Run

```bash
bun install
bun run dev
```
